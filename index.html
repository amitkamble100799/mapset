Q2- Problem: You are given a string (STR) of lenght N. you have to print the count of all alphabet.(using maps)

                
the given string: abcadeecfb



                output: 
                a=2 
                b=2 
                c=2 
                d=1 
                e=2 
                f=1 
            

                ans: syntax:
                var str = "abcadeecfb";
                const ab = new Map();
                for (var alphabet of str) {
                  if (ab.has(alphabet)) {
                 ab.set(alphabet,ab.get(alphabet) + 1);
                  } else {
                    ab.set(alphabet, 1);
                  }
                }
                console.log(ab);
